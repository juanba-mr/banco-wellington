# banco-wellington
Banco Wellington - Aplicación Web en Java
Un sistema integral de gestión bancaria desarrollado con Java (JSP/Servlets), utilizando una arquitectura Modelo-Vista-Controlador (MVC). Este proyecto permite administrar usuarios, cuentas bancarias y transacciones con un backend persistente en MySQL.

Tecnologías Utilizadas
Backend: Java EE (Servlets, JSP, JSTL).

Servidor Web: Apache Tomcat 9.0+.

Base de Datos: MySQL.

Frontend: HTML5, CSS3, JavaScript.

Control de Versiones: Git.

Características Principales
Gestión de Usuarios: Funcionalidades de administrador para crear, modificar y dar de baja clientes.

Operaciones de Cuenta: Soporte para múltiples tipos de cuenta (Ahorro/Corriente) con seguimiento de saldo.

Sistema de Transacciones: Lógica para depósitos, retiros y transferencias entre cuentas.

Validación de Datos: Validación en el backend para operaciones financieras seguras.
